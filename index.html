🔑 Connect to GitHub

Enter your GitHub username and a Personal Access Token (PAT) to load your repositories. Your token is stored locally and never sent anywhere except GitHub's API.

Create a token at github.com/settings/tokens with repo (read + write, needed to commit site changes) and read:user scopes. Use public_repo instead of repo if you only manage public repositories.
